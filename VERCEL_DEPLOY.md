# Деплой бота на Vercel (пошагово)

Бот на Vercel работает по **webhook**: Telegram сам шлёт обновления на твой URL. После деплоя нужно один раз указать этот URL в Telegram.

---

## Шаг 1. Подключи проект к Vercel

1. Зайди на [vercel.com](https://vercel.com) и войди в аккаунт (или зарегистрируйся).
2. Нажми **Add New…** → **Project**.
3. Импортируй репозиторий с этим кодом (GitHub / GitLab / Bitbucket) или загрузи папку через **Import** (если репо ещё нет — создай его и запушь код).
4. **Важно:** в настройках проекта укажи **Root Directory** — корень проекта (папка, где лежат `vercel.json`, `api/`, `bot/`).
5. Не нажимай пока Deploy — сначала добавь переменные (следующий шаг).

Когда будешь готов, напиши «готово» или «шаг 2» — дам следующий шаг.
